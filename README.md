# 🚗 Speed Champion

**Speed Champion** — это аркадная гоночная игра, разработанная на **C# Windows Forms** с использованием псевдо-3D рендеринга дороги в стиле классических игр (*OutRun*, *Need for Speed SE*).

Проект демонстрирует создание гоночной игры без использования игровых движков, только средствами **.NET Framework** и **System.Drawing**.

---

## 📸 Возможности

* 🛣️ Псевдо-3D трасса с перспективой
* 🌄 Холмы и плавные повороты
* 🌳 Генерация деревьев, столбов и рекламных щитов
* 🚗 Управление автомобилем
* 🤖 Машины соперников (AI)
* 💥 Простая система столкновений
* ⚡ Нитро-ускорение
* 🏁 Подсчет кругов
* 📊 Игровой HUD
* 🎮 Частота обновления около 60 FPS (Timer 16 мс)

---

## 🛠 Используемые технологии

| Клавиша   | Действие            |

## ⚙ Основные возможности проекта

### Генерация трассы

Трасса состоит из отдельных сегментов.

Каждый сегмент хранит:

* координаты;
* ширину дороги;
* высоту;
* угол поворота;
* декоративные объекты.

Используются синусоидальные функции для создания естественных холмов и заранее заданные участки с поворотами.

---

### Псевдо-3D

Отображение реализовано через перспективную проекцию:

* дальние сегменты становятся уже;
* ближние — шире;
* создаётся эффект глубины без использования 3D-графики.

---

### Искусственный интеллект

Соперники:

* двигаются с собственной скоростью;
* занимают разные полосы;
* слегка реагируют на положение игрока;
* могут сталкиваться с автомобилем игрока.

---

### Игровая физика

Реализованы:

* ускорение;
* торможение;
* инерция;
* ограничение скорости;
* нитро;
* потеря скорости при столкновении.
